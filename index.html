Introduction

Welcome to the documentation for Skyrim!

Skyrim was originally released in 2011. It has been released for PC, in addition to Xbox and PlayStation consoles.

Over the years, many expansion packs, mods, and editions have been developed for Skyrim. this documentation will only describe the base game.

Creating a Character

Creating a character on Skyrim is relatively easy. You create your character from four major attributes:

  • Race
  • Gender
  • Appearance
  • Name

Some more information about the attributes that are required to create your character:

  • You need to pick a race for your character. The available races in Skyrim are:
    • Nord (most popular)
    • Argonian
    • Breton
    • Dark Elf
    • High Elf
    • Imperial
    • Khajiit
    • Orc
    • Redguard
    • Wood Elf
  • You can only choose two genders in Skyrim: male or female.
  • You can choose from multiple appearance options:
    • Weight/Muscles
    • Skin color
    • Hair color and style
    • War paint and/or tattoos
    • Dirt
    • Scars
    • Beard (for male characters only)
    • Jewelry
    • Feathers/Horn (for Argonian characters only)
  • Finally, you need to create a name for your character.
Storyline

Skyrim is the fifth game in the Elder Scrolls video game series. This is a new story, based in a new area.

Skyrim takes place nearly 200 years after the Oblivion Crisis. Skyrim, the Nords' homeland, is in the midst of a civil war. You are the Dragonborn and you must face these threats, along with other dangers in the environment, to save Skyrim.

Troubleshooting

Skyrim, like many video games, does contain some glitches and bugs. Some of these glitches and bugs are easily fixed. Please see the list below for the most common glitches bugs and how to fix them.

  • Getting stuck
    • If you find that you're stuck in between objects, first try fast traveling to a new location.
    • Use the console command tcl to turn off collision detection, and then moving through the object. Don't forget to use tcl to turn collision detection back on.
  • Sounds
    • Sometimes, the combat music doesn't load correctly or continues to play after the fight has ended. If this happens, try the console command removemusic muscombat.
  • Quest item status
    • If a regular item gets "quest item" status attached to it, try the console command player.removeitem to remove the affected items and then enter console command player.additem to re-add the items.
